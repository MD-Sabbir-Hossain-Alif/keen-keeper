# 📌 KeenKeeper — Don't Lose Your Friendships

KeenKeeper is a modern web app designed to help you maintain and strengthen your friendships. It reminds you to stay connected, track interactions, and visualize your social relationships over time.

## 🚀 Live Preview

🔗 [Visit KeenKeeper](https://keen-keeper-iota-nine.vercel.app/)

## 🧠 Project Overview

In today’s busy world, it’s easy to lose touch with friends. KeenKeeper helps you stay mindful of your relationships by tracking when you last connected and providing insights through visual data.

## 🛠️ Technologies Used

- **React.js + Next.js** – Frontend development
- **App Router (Next.js)** – Navigation handling
- **Tailwind CSS + DasiyUI** – Styling and responsive UI
- **React Toastify** – Toast visualization (like alart)
- **Recharts** – Data visualization and charts

## ✨ Key Features

1. 📅 **Smooth User Experience**  
   Built with Next.js App Router, featuring both **Server Components** and **Client Components** on the same page for optimal performance. Next.js makes the app faster than traditional React with its excellent file-based routing system.

2. 📊 **Filtering Interactions**  
   Track different types of interactions (Call, Text, Video Call, etc.) with friends. All interactions are added to the Timeline using **Context API**. Users can easily filter the timeline data using a dropdown menu.

3. 🔔 **Visual Statistics**  
   View all your interactions through beautiful **Pie Charts** powered by Recharts, providing a clear graphical representation of your social activity.
